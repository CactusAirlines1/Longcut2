Instructions

To set up your ID for the website, follow these steps:

  1. Locate the comment at the top of the shortcut. It points to a text box where you should enter your ID.
  2. Find the "Get Item from List" action. Change its index to "2."
  3. Run the shortcut. This process assigns a date, which might be slightly off by a few hours but is generally accurate.

In the video tutorial, a "Stop This Shortcut" action was added before the web request to prevent creating a new ID. Do not add this action to your shortcut; it will prevent it from working properly. Also in the video, I used the ID "FakeID". The ID you use does not matter.
