Instructions for Changing the ID for the Website (Recommended to do this right away)

First, there's a comment right at the top, it has an arrow which points to a text box. That's where you want to put your ID. There is a list, look for the "Get Item from List" action, it should be at index 1. Change it to 2. Then, run the shortcut. All this is just to get a date. The date it will show might be slightly different than what it is actually supposed to be, usually by a few hours. No big deal. It's not all that important, but it's a nice detail.

Below will be a video showing the entire process. It doesn't take long. In the video, there is an additional action that I put in so that I didn't have to clear the ID that would be created. This action is the "Stop This Shortcut" action, which prevents the shortcut from creating another ID because it is before the web part. This action is not in the shortcut linked here (I removed it). Do not add this action in the shortcut. It will not work.
