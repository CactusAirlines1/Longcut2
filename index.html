Instructions for Changing the ID for the Website

To properly set up Longcut, follow these steps:

  1. Locate the comment at the top of the shortcut. It has an arrow pointing to a text boxâ€”this is where you should enter your ID.
  2. Find the "Get Item from List" action, which is initially set to index 1. Change this index to 2.
  3. Run the shortcut. This step is necessary to generate the date.

Note: In the demonstration video, a "Stop This Shortcut" action was temporarily added before the web request step to prevent overwriting an existing ID. This action has been removed from the final shortcut. Do not add this action to the shortcut, as it will prevent it from working correctly.
