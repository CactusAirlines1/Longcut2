Instructions for Changing the ID for the Website (Recommended to do this right away)

First, there’s a comment right at the top of the shortcut, it has an arrow pointing to a text box. That’s where you want to put your ID. Then, locate the "get item from list" action and change the index to "2." Once this is done, you can run the shortcut.

It’s important to note that the date the shortcut will display might be a bit different from the actual date, usually by a few hours, but this detail is not crucial. If you don’t change the index, the shortcut will generate a date based on your device name.

In the video, there is an additional action that stops the shortcut from creating another ID. This action is the "stop this shortcut" action, which was added to the shortcut. However, do not add this action to the shortcut yourself, as it will prevent it from working correctly.
